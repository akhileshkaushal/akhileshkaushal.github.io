Overview
----------------------------------------------------------

This documentation provides a complete overview of the methylation analysis pipeline using Illumina Infinium MethylationEPIC arrays. It includes setup, preprocessing, statistical modeling, gene-region annotation, visualization, and example use cases with regulatory interpretation.

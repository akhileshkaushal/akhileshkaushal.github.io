Output Summary
==============

This section describes the structure and contents of the output files generated by the RNA-seq pipeline. Each analysis is comparison-specific and organized in a modular directory layout.

Directory Structure
-------------------

The pipeline creates a unique subdirectory under `outputs/` (or your `base_output_path`) for each group comparison. All plots, tables, logs, and enrichment results are stored here.

Example Layout:

.. code-block:: text

   Analysis/
   ├── logs/
   │   ├── Group1_vs_Group2_log.txt
   ├── Group1_vs_Group2/
   │   ├── Group1_vs_Group2_deseq2_results.csv
   │   ├── Group1_vs_Group2_limma_results.csv
   │   ├── Group1_vs_Group2_upregulated_genes.csv
   │   ├── Group1_vs_Group2_downregulated_genes.csv
   │   ├── Group1_vs_Group2_ranked_genes.rnk
   │   ├── Group1_vs_Group2_volcano_plot.tiff
   │   ├── Group1_vs_Group2_heatmap.png
   │   ├── Group1_vs_Group2_KEGG/
   │   │   ├── hsa04610.Group1_vs_Group2_hsa04610.png
   │   │   └── ...
   │   ├── Group1_vs_Group2_unmapped_symbols.csv
   │   └── Group1_vs_Group2_ORA_results.xlsx

Output File Descriptions
-------------------------

+-------------------------------+--------------------------------------------------------------+
| File                          | Description                                                  |
+===============================+==============================================================+
| *_deseq2_results.csv          | DESeq2 output with log2FC, p-value, and adjusted p-values    |
| *_limma_results.csv           | Limma output with moderated t-statistics and p-values        |
| *_upregulated_genes.csv       | Genes with log2FC > 0.3 and padj < 0.25                      |
| *_downregulated_genes.csv     | Genes with log2FC < -0.3 and padj < 0.25                     |
| *_ranked_genes.rnk            | Ranked genes (by log2FC) for GSEA                           |
| *_volcano_plot.tiff           | Volcano plot highlighting DEGs                              |
| *_heatmap.png                 | Clustered heatmap of top DEGs                               |
| *_ORA_results.xlsx            | ORA results (multiple tabs for gene sets and directions)     |
| *_KEGG/*.png                  | KEGG pathway overlays showing mapped gene expression         |
| *_unmapped_symbols.csv        | Genes unmapped during SYMBOL ➜ ENTREZ conversion             |
+-------------------------------+--------------------------------------------------------------+

Log Files
---------

For every comparison, a log file tracks:
- Execution time and version info
- Loaded files and sample stats
- Errors and warnings (if any)
- Summary of generated output

This is helpful for troubleshooting and ensuring reproducibility.

Notes
-----

- All outputs are grouped by the `name` value in `comparisons.json`
- File naming conventions follow: `<comparison>_<task>.ext`
- Paths are relative to `base_output_path`, which can be customized
- Output formats are consistent across comparisons for easy automation

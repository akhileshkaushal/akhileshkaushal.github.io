Transcriptomic-Pipeline

A pipeline for preprocessing and differential analysis of RNA sequencing data generated via NextSeq or HiSeq.

Transcriptomic RNA-Seq STAR Kallisto Subread DESeq2 EdgeR

Overview

This pipeline provides a comprehensive framework for transcriptomic differential expression and enrichment analysis using RNA-Seq count data. It uses DESeq2 and limma for statistical modeling, and clusterProfiler and fgsea for functional enrichment. Outputs include enhanced volcano plots, heatmaps, and KEGG pathway visualizations.

Features

  • Reads configuration from a structured JSON file
  • Supports multiple comparisons across groups
  • Handles sex as a covariate when mixed genders are present
  • Performs DE analysis using both DESeq2 and limma-voom
  • Generates enhanced volcano plots and heatmaps
  • Saves DEGs (up/down-regulated), RANK files for GSEA
  • Performs batch ORA and GSEA using custom GMT files
  • Pathview visualizations of KEGG pathways with mapped log2 fold changes
  • Extensive logging and modular design for future extension

Tools & Libraries Used

  • R, Rscript, DESeq2, limma, edgeR for statistical modeling
  • EnhancedVolcano, pheatmap for visualization
  • clusterProfiler, fgsea for enrichment analysis
  • pathview for KEGG visualizations
  • jsonlite, openxlsx, dplyr for data processing

Input Requirements

  • A tab-delimited RNA-seq count matrix (samples as columns, genes as rows)
  • A comparisons.json configuration file specifying group comparisons
  • Optional: GSEA/ORA config file (gsea_config.json) and pathway .gmt files

Output Summary

  • Differential gene expression tables
  • Up/downregulated gene lists
  • Volcano plots and heatmaps
  • ORA and GSEA result files
  • Pathway visualizations from pathview

Installation - RNA-seq Pipeline

Installation Guide

Prerequisites

Before running the pipeline, ensure the following software are installed:

  • R (≥ 4.0)
  • Rtools (for Windows users)
  • Git (for cloning the repository)

Clone the Repository

git clone https://github.com/akhilesh-kaushal/rnaseq-pipeline.git
cd rnaseq-pipeline

Install Required R Packages

Open R or RStudio and run the following:

# Install BiocManager (if not already installed)
install.packages("BiocManager")

# Core packages
BiocManager::install(c(
  "DESeq2", "limma", "EnhancedVolcano", "pheatmap", 
  "clusterProfiler", "org.Hs.eg.db", "pathview", "fgsea", 
  "enrichplot", "openxlsx", "ggrepel"
))

# CRAN dependencies
install.packages(c("jsonlite", "ggplot2", "dplyr", "tibble", "tools", "rmarkdown"))

⚙️ Project Directory Setup

  • Create a directory to store comparison outputs (e.g., outputs/)
  • Edit comparisons.json and gsea_config.json to define your experimental groups
  • Ensure your HTSeq or featureCounts matrix is correctly formatted

Running the RNA-seq Pipeline

Running the RNA-seq Pipeline

Main Entry Script: main.R

This script orchestrates all modules – data loading, differential expression (DE), visualization, and enrichment.

📝 Step 1: Configure the Comparison JSON

Edit comparisons.json with your sample groups:

{
  "counts_file": "path/to/count_matrix.tsv",
  "base_output_path": "outputs",
  "comparisons": [
    {
      "name": "Case_vs_Control",
      "group1": ["Sample_1", "Sample_2"],
      "group2": ["Sample_3", "Sample_4"],
      "sex_groups": ["M", "M", "F", "F"]
    }
  ]
}

Step 2: Launch the Pipeline

# In terminal or R console
Rscript main.R

⚙️ What Happens Internally

  • 🔹 Loads configuration from comparisons.json
  • 🔹 Reads count matrix and filters low-expression genes
  • 🔹 For each comparison:
    • Runs DESeq2 and limma
    • Saves DE tables, rank files
    • Generates volcano plot and heatmap
    • Optionally performs KEGG Pathview and GSEA/ORA

RNA-seq Visualization

RNA-seq Visualization Module

🔬 Purpose

This module generates plots for easier interpretation of transcriptomic changes across conditions:

  • Volcano plot – significance vs fold-change
  • Heatmap – top differentially expressed genes
  • Pathview KEGG pathway overlays

📊 Volcano Plot (DESeq2)

Uses EnhancedVolcano to highlight up/down-regulated genes.

EnhancedVolcano(res_df,
  lab = rownames(res_df),
  x = 'log2FoldChange',
  y = 'padj',
  pCutoff = 0.25,
  FCcutoff = 0.3,
  title = 'Volcano: Case_vs_Control',
  legendPosition = 'none'
)

🧊 Heatmap of Top DEGs

Top 50 up/down genes selected by log2FoldChange and plotted using pheatmap().

heatmap_data <- counts_matrix[top_genes, samples]
pheatmap(heatmap_data, scale = "row", main = "Top DEGs")

🧬 KEGG Pathview

Uses pathview to overlay gene expression onto KEGG pathways. Requires internet or local XML files.

pathview(
  gene.data = gene_list,
  pathway.id = "hsa04610",
  species = "hsa",
  out.suffix = "Case_vs_Control"
)

Example Output

Check your outputs/ directory for .png Pathview diagrams.

  • Plots are saved as TIFF/PNG under each comparison folder
  • Volcano/Heatmap require sufficient DEGs to render

RNA-seq Enrichment Analysis

Functional Enrichment Analysis

Overview

This module analyzes biological meaning behind differentially expressed genes using:

  • ORA (Over Representation Analysis) via clusterProfiler::enricher()
  • GSEA (Gene Set Enrichment Analysis) via fgsea::fgsea()

ORA: Over Representation Analysis

ORA identifies significantly enriched pathways in up/down gene sets using GMT files.

result <- enricher(
  gene = gene_list,
  TERM2GENE = gmt_df,
  pvalueCutoff = 0.05,
  qvalueCutoff = 0.2
)

GSEA: Gene Set Enrichment Analysis

Ranks genes by log2FoldChange and tests for enrichment of ranked genes in each pathway.

fgseaRes <- fgsea(
  pathways = gmt_list,
  stats = ranked_genes,
  minSize = 50,
  maxSize = 500
)

Random Walk Plot (Enrichment Plot)

For top enriched gene sets (based on NES and padj), the enrichment curve is plotted.

plotEnrichment(pathways[[1]], ranked_genes) + ggtitle("Pathway Enrichment")

📁 Output Files

  • ORA_results.xlsx: Sheet per gene set & direction (e.g., KEGG_HYPER)
  • *_dotplot.tiff: Top 15 enriched terms
  • gsea_results.csv: Pathway NES, p-values

🧩 Inputs

  • ranked_genes.rnk for GSEA
  • upregulated_genes.csv / downregulated_genes.csv for ORA
  • Gene set collections in .gmt format (KEGG, GO BP, Hallmarks, etc.)

⚠️ Tips

  • Use curated gene sets relevant to your study
  • Review NES, padj, and pathway size carefully
  • Ensure gene names match the GMT format (SYMBOL or ENTREZ)

KEGG Pathway Overlay

🧬 KEGG Pathway Overlay

Overview

This module uses the pathview R package to overlay differential expression results onto KEGG pathways.

⚙️ Workflow

  1. Filter DEGs using log2FC and FDR thresholds
  2. Map SYMBOL ➜ ENTREZ using org.Hs.eg.db
  3. Match with KEGG pathway IDs
  4. Generate visual overlays with pathview()

Code Example

library(pathview)
library(clusterProfiler)
library(org.Hs.eg.db)

# Convert SYMBOL to ENTREZ
entrez_map <- bitr(gene_symbols, fromType = "SYMBOL", toType = "ENTREZID", OrgDb = org.Hs.eg.db)

# Build gene list with log2FC values
res_df <- read.csv("group_deseq2_results.csv", row.names = 1)
mapped <- intersect(entrez_map$SYMBOL, rownames(res_df))
gene_list <- res_df[mapped, "log2FoldChange"]
names(gene_list) <- entrez_map$ENTREZID[match(mapped, entrez_map$SYMBOL)]

# Generate Pathview for KEGG ID
pathview(gene.data = gene_list,
         pathway.id = "hsa04610",
         species = "hsa",
         out.suffix = "group_hsa04610")

📂 Output Files

  • *.png: KEGG pathway overlays with gene expression color scale
  • *_unmapped_symbols.csv: Unmatched genes (optional)

Example Pathways

  • hsa04610 – Complement & coagulation cascades
  • hsa04611 – Platelet activation
  • hsa04060 – Cytokine-cytokine receptor interaction
  • hsa05200 – Pathways in cancer

Output Example

Notes

  • Gene ID format must match pathway data – SYMBOL ➜ ENTREZ recommended
  • Supports pathway caching or pre-download for offline use
  • Color gradient = expression values (log2FC)

Background
